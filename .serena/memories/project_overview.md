Xponge is a Python package for molecular simulation preprocessing and postprocessing, aimed at SPONGE but also able to handle common trajectory/structure formats. Main package code is under Xponge/. CLI entrypoints are defined in Xponge/__main__.py and exposed via `python -m Xponge` or `Xponge` after install. Relevant post-analysis code for trajectory CLI is in Xponge/tools/traj_analysis.py. Repository root also contains docs/, README.md, pyproject.toml, setup.py, and built-in unittest-style tests under Xponge/tools/unittests/.
