Useful commands: `python -m Xponge -h` or `Xponge -h` for CLI help; `python -m Xponge traj ...` / `Xponge traj ...` for trajectory analysis; `Xponge test --do base -o test --verbose 1` for basic packaged tests; `rg -n PATTERN .` and `rg --files` for fast code search; standard Linux utilities (`ls`, `find`, `sed -n`, `nl -ba`) are useful. Installation/docs mention `python setup.py install`, and dependencies are defined in pyproject.toml / env files.
